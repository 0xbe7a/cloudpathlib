32 https://docs.microsoft.com/en-us/python/api/azure-storage-blob/azure.storage.blob.blobserviceclient?view=azure-python). 

33 Supports the following authentication methods of `BlobServiceClient`. 

34 

35 - Environment variable `""AZURE_STORAGE_CONNECTION_STRING"` containing connecting string 

36 with account credentials. See [Azure Storage SDK documentation]( 

37 https://docs.microsoft.com/en-us/azure/storage/blobs/storage-quickstart-blobs-python#copy-your-credentials-from-the-azure-portal). 

38 - Account URL via `account_url`, authenticated either with an embedded SAS token, or with 

39 credentials passed to `credentials`. 

40 - Connection string via `connection_string`, authenticated either with an embedded SAS 

41 token or with credentials passed to `credentials`. 

42 - Instantiated and already authenticated [`BlobServiceClient`]( 

43 https://docs.microsoft.com/en-us/python/api/azure-storage-blob/azure.storage.blob.blobserviceclient?view=azure-python). 

44 

45 If multiple methods are used, priority order is reverse of list above (later in list takes 

46 priority). 

47 

48 Args: 

49 account_url (Optional[str]): The URL to the blob storage account, optionally 

50 authenticated with a SAS token. See documentation for [`BlobServiceClient`]( 

51 https://docs.microsoft.com/en-us/python/api/azure-storage-blob/azure.storage.blob.blobserviceclient?view=azure-python). 

52 credential (Optional[Any]): Credentials with which to authenticate. Can be used with 

53 `account_url` or `connection_string`, but is unnecessary if the other already has 

54 an SAS token. See documentation for [`BlobServiceClient`]( 

55 https://docs.microsoft.com/en-us/python/api/azure-storage-blob/azure.storage.blob.blobserviceclient?view=azure-python) 

56 or [`BlobServiceClient.from_connection_string`](

58 connection_string (Optional[str]): A connection string to an Azure Storage account. See 

59 [Azure Storage SDK documentation](

61 blob_service_client (Optional[BlobServiceClient]): Instantiated [`BlobServiceClient`](

107 def _is_file_or_dir(self, cloud_path: AzureBlobPath) -> Optional[str]: 

108 # short-circuit the root-level container 

109 if not cloud_path.blob: 

110 return "dir" 

111 

112 try: 

113 self._get_metadata(cloud_path) 

114 return "file" 

115 except ResourceNotFoundError: 

116 prefix = cloud_path.blob 

117 if prefix and not prefix.endswith("/"): 

118 prefix += "/" 

119 

120 # not a file, see if it is a directory 

121 container_client = self.service_client.get_container_client(cloud_path.container) 

122 

123 try: 

124 next(container_client.list_blobs(name_starts_with=prefix)) 

125 return "dir" 

126 except StopIteration: 

127 return None

132 def _list_dir( 

133 self, cloud_path: AzureBlobPath, recursive: bool = False 

134 ) -> Iterable[AzureBlobPath]: 

135 container_client = self.service_client.get_container_client(cloud_path.container) 

136 

137 prefix = cloud_path.blob 

138 if prefix and not prefix.endswith("/"): 

139 prefix += "/" 

140 

141 yielded_dirs = set() 

142 

143 # NOTE: Not recursive may be slower than necessary since it just filters 

144 # the recursive implementation 

145 for o in container_client.list_blobs(name_starts_with=prefix): 

146 # get directory from this path 

147 for parent in PurePosixPath(o.name[len(prefix) :]).parents: 

148 

149 # if we haven't surfaced thei directory already 

150 if parent not in yielded_dirs and str(parent) != ".": 

151 

152 # skip if not recursive and this is beyond our depth 

153 if not recursive and "/" in str(parent): 

154 continue 

155 

156 yield self.CloudPath(f"az://{cloud_path.container}/{prefix}{parent}") 

157 yielded_dirs.add(parent) 

158 

159 # skip file if not recursive and this is beyond our depth 

160 if not recursive and "/" in o.name[len(prefix) :]: 

161 continue 

162 

163 yield self.CloudPath(f"az://{cloud_path.container}/{o.name}")

187 def _remove(self, cloud_path: AzureBlobPath) -> None: 

188 if self._is_file_or_dir(cloud_path) == "dir": 

189 blobs = [b.blob for b in self._list_dir(cloud_path, recursive=True)] 

190 container_client = self.service_client.get_container_client(cloud_path.container) 

191 container_client.delete_blobs(*blobs) 

192 elif self._is_file_or_dir(cloud_path) == "file": 

193 blob = self.service_client.get_blob_client( 

194 container=cloud_path.container, blob=cloud_path.blob 

195 )

197 blob.delete_blob()
